== Description ==

[Demo](https://tishonator.com/plugins/immersive-slider)

ImmersiveSlider plugins allows you to insert and configure a Responsive jQuery Slider into your WordPress site as a shortcode. 

= Shortcodes: =

Insert `[immersive-slider]` on a page to display a slider with data configured in Admin Panel -> Left Menu -> ImmersiveSlider Page. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `immersive-slider` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Admin Panel -> 'Immersive Slider' -> fill Slider fields
4. insert the shortcode [immersive-slider] to display Slider to any page or post
5. Enjoy!

== Pro Features ==

 - Configure Up to 10 Different Sliders
 - Insert Up to 10 Slides per Slider
 - Color Options: Slider Background, Title and Text, Link and Link Hover.
 - Sliding Settings: Sliding Delay, Pagination
 - Sliding Effects: slide, bounce, fade, slideUp, or bounceUp.
